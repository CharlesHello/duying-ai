# 渡萤 - AI聊天助手

## 产品简介
渡萤是一款支持多模型切换的AI聊天助手，帮助用户高效完成对话、创作、代码开发等任务。

## 核心功能
- 🤖 **多模型支持**：Claude、GPT、Gemini等主流大模型
- 💬 **对话管理**：历史记录、搜索、导出
- 📁 **文件解析**：支持PDF、Word、Excel等格式上传
- 🎙️ **语音交互**：语音输入与TTS朗读

## 技术栈
- 前端：React + TypeScript
- 后端：Spring Boot
- 数据库：PostgreSQL
